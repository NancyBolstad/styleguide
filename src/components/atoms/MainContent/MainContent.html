This is a component that can be used to wrap the main content of a page to make it responsive and give the page a bottom margin.

If you add the main-content--padding-top CSS class the main-content element will be padded on top. This is solved with a modifier class because not all elements within the main-content element should have padding on top.

Add main-content--bg-color-white if you want the background-color to be white.
